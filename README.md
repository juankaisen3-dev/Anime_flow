# 🎧 Anime FLOW – Plateforme de streaming d’anime

**Anime FLOW**, c’est notre projet à **Daniel_Tech** et **Roberto_SkyTech's**.  
Nous sommes  devs fullstack, et on a décidé de créer un site de streaming d’anime moderne, rapide et agréable à utiliser.

---

## 🧠 Pourquoi Anime FLOW ?

On voulait un player qui déchire, un catalogue bien organisé, et zéro prise de tête.  
On gère l’intégralité du projet ensemble : du CSS à la BDD, en passant par l’hébergement vidéo.

---

## 🛠️ Stack technique (choisie ensemble)

### Backend
- Node.js + Express (ou FastAPI si on change d’avis)
- MongoDB / PostgreSQL (selon les besoins)
- API REST + WebSocket (pour les notifs)

### Frontend
- React / Next.js (App Router)
- TailwindCSS
- Lecteur : Video.js + HLS.js

### Infra / Outils
- Git + GitHub
- Docker + docker-compose
- Nginx + Cloudflare
- vercel / render pour l’hébergement
